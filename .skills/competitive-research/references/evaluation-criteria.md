# Criterios de Evaluación de Competidores

## Dimensiones de Análisis

Cada competidor se evalúa en estas 8 dimensiones, con puntaje de 1-5:

### 1. Cobertura Funcional (¿Qué hace?)
- 1: Solo POS básico
- 3: POS + stock + reportes
- 5: Suite completa (POS, stock, RRHH, servicios, multi-sucursal, analytics)

### 2. Experiencia de Usuario (¿Es fácil de usar?)
- 1: Interfaz anticuada, requiere capacitación
- 3: Funcional, curva de aprendizaje moderada
- 5: Intuitivo, un kiosquero lo entiende en minutos

### 3. Accesibilidad Tecnológica (¿Desde dónde se usa?)
- 1: Solo desktop con instalación
- 3: Web responsive o app mobile
- 5: PWA + offline + funciona en cualquier dispositivo

### 4. Precio / Valor (¿Vale lo que cuesta?)
- 1: Caro para lo que ofrece
- 3: Precio justo, estándar del mercado
- 5: Excelente relación precio-valor o freemium generoso

### 5. Adaptación al Mercado Local (¿Entiende Argentina/Latam?)
- 1: Producto genérico internacional sin localización
- 3: Tiene moneda local y algo de integración
- 5: AFIP, Mercado Pago, QR, cultura del kiosco, proveedores locales

### 6. Escalabilidad (¿Crece con el negocio?)
- 1: Para un solo local, difícil escalar
- 3: Multi-local posible pero limitado
- 5: Pensado para cadenas, con roles, permisos, consolidación

### 7. Diferenciación (¿Qué tiene de único?)
- 1: Commodity, igual a todos
- 3: Algún diferencial menor
- 5: Features únicas que nadie más ofrece

### 8. Ecosistema e Integraciones
- 1: Sistema cerrado, sin integraciones
- 3: Algunas integraciones básicas
- 5: API abierta, marketplace de integraciones, conecta con todo

## Señales de Alerta en Competidores

Cosas que indican que un competidor es débil o que hay oportunidad:
- Web sin actualizar hace más de un año
- Sin versión mobile
- Requiere instalación en Windows
- Sin soporte para servicios virtuales (SUBE, recargas)
- Sin multi-sucursal
- Pricing opaco (hay que "solicitar demo")
- Reviews negativos sobre soporte técnico
- Sin facturación electrónica AFIP

## Señales de Amenaza

Cosas que indican que un competidor es fuerte:
- Crecimiento visible en redes/reviews
- Actualizaciones frecuentes
- Integraciones con Mercado Pago, AFIP, bancos
- Versión mobile nativa con buenas reviews
- Comunidad activa de usuarios
- Precio agresivo con buen producto
